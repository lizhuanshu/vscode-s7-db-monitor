# Contributing

Thanks for helping improve S7 DB Monitor.

## Development Setup

```powershell
npm install
npm test
```

Press `F5` in VS Code to launch the Extension Development Host.

## Pull Requests

- Keep changes focused and easy to review.
- Add or update tests for parser, decoder or communication behavior when possible.
- Run `npm test` before opening a pull request.
- Use CRLF line endings on Windows, as configured by `.gitattributes`.

## Issues

When reporting parser issues, include a minimal `.db` snippet and describe the expected offsets or hierarchy.
