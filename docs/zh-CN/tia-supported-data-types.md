# TIA Portal 支持的数据类型

[English](../tia-supported-data-types.md)

TIA Portal 数据类型定义了值在 PLC 存储区中的表示方式。用户程序可以使用预定义数据类型，也可以将它们组合成 PLC 数据类型，例如 UDT。

本插件重点支持可以从 DB 存储区显示和解析的数据：

- 基本数据类型：二进制数、整数、浮点数、日期时间、字符和字符串。
- 复杂数据类型：数组、结构和字符串。
- PLC 数据类型：同文件中的 UDT 定义。

指针、参数、系统和硬件数据类型可能出现在 TIA Portal 项目中，但当前插件不会把它们作为可监控的标量值进行解码。

实际可用类型取决于 PLC 系列、CPU 固件，以及 DB 是否使用优化访问。

## 二进制数

| 类型 | S7-300/400 | S7-1200 | S7-1500 | 监控支持 |
| --- | --- | --- | --- | --- |
| `Bool` | 是 | 是 | 是 | 是 |
| `Byte` | 是 | 是 | 是 | 是 |
| `Word` | 是 | 是 | 是 | 是 |
| `DWord` | 是 | 是 | 是 | 是 |
| `LWord` | 否 | 否 | 是 | 是 |

## PLC 数据类型

| 类型 | S7-300/400 | S7-1200 | S7-1500 | 监控支持 |
| --- | --- | --- | --- | --- |
| UDT | 是 | 是 | 是 | 同文件中的 `TYPE "UDT"` 定义 |
| `Struct` | 是 | 是 | 是 | 是 |
| `Array[...] of ...` | 是 | 是 | 是 | 是 |

## 其他 TIA 数据类型组

TIA Portal 还包含指针类型、参数类型、系统数据类型和硬件数据类型。例如 `Variant`、`Pointer`、`Any`、`Timer`、`Counter`、`Block_FC`、`Block_FB`、`IEC_Timer`、`TCON_Param`、`HW_Device`、`OB_Any` 和 `DB_Any`。

这些类型在 PLC 程序和接口中很有用，但不属于当前 DB 监控器的解码范围。如果它们出现在导出的 DB 文件中，解析器应该报告为不支持，而不是猜测其内存布局。
